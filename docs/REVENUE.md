# Revenue Strategy

## Channels

### 1. Cloud Credit Affiliate Links (Primary)
- AWS Activate, GCP credits, Azure for startups
- Link from "Getting Started" recommendations
- Typical commission: $50–150 per qualified signup

### 2. API Platform Referrals
- OpenAI, Anthropic referral programs
- Contextual placement alongside pricing data
- Commission varies by platform

### 3. Display Ads (Secondary)
- Google AdSense once traffic exceeds 1K/day
- Non-intrusive placement (sidebar, between sections)
- Expected CPM: $2–5 for developer audience

## Traffic Projections

| Month | Monthly Visits | Revenue Est. |
|-------|---------------|-------------|
| 1–2 | 100–500 | $0 |
| 3–4 | 1K–5K | $50–150 |
| 5–6 | 5K–15K | $150–400 |
| 7–12 | 15K–50K | $300–800 |

## Growth Levers
- Weekly pricing updates drive return visits
- SEO blog content for comparison queries
- Social sharing of calculator results
- API for developers to embed pricing widgets
